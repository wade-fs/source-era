# 連貫性規範

## 設定一致性

- 源質構成比例一旦在文中確定，後續章節不可更改
- 角色職業等級不可無故跳升（必須有合成成功、任務完成等明確事件）
- 配方成功率影響結果，不可每次都剛好成功

## 伏筆管理

- 所有伏筆必須在文中用 `<!-- FORESHADOW: F編號 描述 -->` 標記
- 已標記的伏筆揭露時，在當章末尾加 `<!-- REVEALED: F編號 -->` 標注
- 不可無預警揭露從未暗示過的重大真相

## 時間線

- 每章應有明確的遊戲內時間（Day X, 上午/下午/夜間）
- 林墨的體力、材料存量需符合前章狀態
- 合成消耗的源質會減少背包存量，不可憑空補充

## NPC 行為一致性

- 老陳：永遠不主動說破答案，只給暗示
- 暗刃七：行事低調，不在公開場合暴露目的
- 鐵臂：話少，但說出來的都是關鍵資訊
