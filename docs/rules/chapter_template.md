# 章節寫作模板

章節標題格式固定為：`### 第 N 章【標題】`
輸出直接從標題開始，不加前言或元資訊。

---

## 結構

### 開場（約 500 字）
- 遊戲內時間、地點、當下狀態
- 林墨的初始目標或行動

### 解析段落（約 800～1200 字）
- 林墨看到的源質數據（用系統框格式呈現）
- 分析過程：數據 → 推論 → 行動方向
- 這是本作區別於一般網遊小說的核心，不可省略

### 主體事件（約 1500 字）
- 合成嘗試 / 探索 / 與 NPC 或玩家互動
- 至少一個意料之外的發現或轉折

### 收尾（約 400 字）
- 本章問題的暫時解決或新的懸念
- 林墨的情緒落點（疑惑、興奮、不安等）

### 伏筆標記（章末，不計入正文）
```
<!-- FORESHADOW: F編號 描述 -->
<!-- REVEALED: F編號 -->（如本章揭露了某伏筆）
```

## 系統提示框格式

解析框：
```
【解析結果】材料名稱
  源質A   XX%
  源質B   XX%
  （隱藏）源質C  0.XX%
```

合成框：
```
【合成嘗試】
公式：化學式
成功率：XX%
結果：成功 / 失敗 / 異常
獲得：物品名稱 × 數量
```

成就框：
```
【隱藏成就解鎖】
成就名稱
達成條件：...
獎勵：...
```
