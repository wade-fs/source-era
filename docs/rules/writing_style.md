# 寫作風格規範

## 視角

- 第三人稱有限視角，緊貼林墨主觀感知
- 讀者只能知道林墨知道的事
- 禁止全知敘事、作者評論

## 節奏

- 偏沉穩觀察，不過度追求刺激
- 「發現 → 分析 → 推論」是林墨的思考節奏，寫作節奏也應如此
- 戰鬥不是重點，源質推理才是

## 字數要求

每章正文純中文字數**不得少於 3500 字**。
字數不足時，應補充以下任一項目直到達標：
- 林墨的解析過程（詳細描述看到了什麼源質數據）
- 環境細節（光線、氣味、聲音、材料質感）
- 合成操作的物理過程（不是直接出結果）
- 林墨的推理內心獨白

## 系統文字格式

遊戲系統提示框用以下格式呈現：

```
【系統提示】
發現材料：月牙池池水
解析結果：
  水源質   81%
  靈源質   10%
  光源質    4%
  命運源質  0.01%  ← 僅超限解析可見
```

合成結果：
```
【合成成功】
配方：W5S2P1
純度：87%
獲得：月光感知藥劑 × 1
隱藏成就：月池之謎 · 序章
```

## 禁止事項

- 禁止角色憑空知道不該知道的資訊
- 禁止跳過分析過程直接得出結論
- 禁止人物行為與 canon/characters.md 矛盾
- 禁止新源質出現時不給任何鋪墊
