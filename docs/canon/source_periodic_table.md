## 已知合成規律

```
土(E) + 水(W) → 泥漿材料
火(F) + 風(A) → 爆裂效果
木(L) + 水(W) → 生長系配方
光(P) + 靈(S) → 治療系配方
暗(D) + 死(N) → 腐化系配方
```
