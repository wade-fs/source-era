# 材料資料庫

格式：每種材料列出來源、源質構成（僅【超限解析】可見完整版）、可提煉源質、備注。

---

## 月牙村地區

### 月牙池池水
```
來源：月牙村中央池塘
一般玩家看見：
  W 85%  水源質
  S 10%  靈源質
  P 4%   光源質
  未知成分 1%
林墨（超限解析）看見：
  W 81%  水源質
  S 10%  靈源質
  P 4%   光源質
  Y 0.01% 命運源質  ← 不應存在於此地
  (其餘為純水)
可提煉：水源質、靈源質（夜間純度更高）
備注：命運源質非自然形成，來源待查
<!-- FORESHADOW: F001 月靈源質並非自然形成 -->
```

### 普通樹葉（月牙森林）
```
來源：月牙森林，地面大量掉落
一般玩家看見：
  L 60%  木源質
  W 20%  水源質
  A 15%  風源質
林墨看見：同上（無隱藏源質）
可提煉：木源質、水源質、風源質
```

### 樹皮
```
來源：月牙森林，砍樹獲得
構成：
  L 55%  木源質
  E 30%  土源質
  A 15%  風源質
可提煉：木源質、土源質
```

### 野花
```
來源：月牙森林草地
構成：
  L 40%  木源質
  P 35%  光源質
  W 15%  水源質
  A 10%  風源質
可提煉：光源質（純度中等）、木源質
備注：光源質含量是新手區最高的材料之一
```

## 礦洞地區

### 銅礦石
```
來源：月牙村初級礦洞，第 1～3 層
構成：
  M 70%  金源質
  E 30%  土源質
可提煉：金源質、土源質
```

### 鐵礦石
```
來源：月牙村初級礦洞，第 4～6 層
構成：
  M 80%  金源質
  E 15%  土源質
  T 5%   雷源質（微量）
可提煉：金源質（高純度）、雷源質（微量）
```

### 夜光石（稀有）
```
來源：礦洞第 6 層深處，極低機率
林墨看見：
  P 60%  光源質
  S 35%  靈源質
  Y 5%   命運源質  ← 再次出現
可提煉：光源質（高純度）、靈源質
備注：命運源質二次出現，林墨注意到規律性
<!-- FORESHADOW: F004 命運源質分布有地理規律，指向某處 -->
```
