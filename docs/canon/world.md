# 世界觀設定

## 職業體系

### 合成術士
研究分解、提煉、合成的核心職業。

進階路線：
```
見習合成師 → 合成術士 → 源質學者 → 煉成大師 → 真理煉成者 → 世界編譯者
```

### 礦工
不只是挖礦。高級礦工尋找：高純度源質礦脈、稀有元素礦脈、古代文明遺跡。
礦工是整個源質經濟的上游。

### 魔法術士（道術師）
不是傳統魔法師，更像道士。透過源質排列形成術式。
例：火源質×5 + 風源質×3 = 火鳥術

### 符術師
進階職業。把合成公式刻進符文。例：「聚靈符」本質上是靈源質吸收公式。

## 解析等級系統

玩家看到的不是傳統「等級」，而是**解析等級**，決定能看懂多少源質成分。

| 等級 | 能看見的內容 |
|---|---|
| 新手 | 主要源質（佔比 >30%）|
| 初級 | 次要源質（佔比 >10%）|
| 中級 | 微量源質（佔比 >1%）|
| 高級 | 痕量源質（佔比 >0.1%）|
| 【超限解析】 | 所有成分，含隱藏源質 |

---

## 源質經濟

- 基礎源質：採集自自然材料，價格低廉
- 衍生源質：需要特定提煉工藝，有市場價值
- 靈性源質：稀有，通常只在特殊地點或材料中出現
- 失落源質：理論上不存在於當前遊戲版本，出現即為異常
