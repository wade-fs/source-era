## 主角

### 林墨
```
職業：見習合成師
天賦：【超限解析】
  - 可解析任意物品的完整源質構成，包含隱藏成分
  - 可看見其他玩家無法識別的配方殘片
  - 代價：資訊過載，初期會頭痛、視覺閃爍
性格：沉穩、觀察力強、習慣獨行；對「不合理的資料」有強迫症般的執著
背景：現實世界中是材料工程系學生，習慣用數據思考
目標（卷一）：搞清楚月牙池的命運源質從哪裡來
外型：黑髮、戴著舊款 VR 頭盔（遊戲內顯示為布衣新手裝）
口頭禪：「數據不會騙人。」
```

角色弧線：
```
卷一：從「看見別人看不見的東西」→ 「明白天賦的代價」
卷三：從「獨行研究者」→ 「被迫面對資訊壟斷勢力」
卷五：從「尋找答案的人」→ 「成為答案本身」
```

## NPC

### 月牙村藥劑師 — 老陳
```
職業：退休合成術士（NPC）
功能：卷一的導師型角色，教導林墨基礎合成
秘密：曾是上古文明研究社群成員，知道月牙池的異常但故意不說
台詞風格：說一半留一半，喜歡用反問
```

### 礦工工會長 — 鐵臂
```
職業：高級礦工（NPC）
功能：提供礦洞深層資訊，後期成為重要情報來源
秘密：他的義肢是用上古合金製作的，成分包含第四環源質
```
