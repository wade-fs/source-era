# 第三卷：失落文明

## 卷主題

世界曾存在超級文明——而他們的知識並沒有真正消失。

## 舊理論 → 新理論

```
源質週期表是自然形成的（無人設計，規律是宇宙本質）
↓ 被推翻
週期表是人造知識（第一煉成者文明刻意設計並封印第四環）
```
