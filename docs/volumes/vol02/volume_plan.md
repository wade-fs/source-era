# 第二卷：礦脈深層

## 卷主題

源質並非自然生成。

## 舊理論 → 新理論

```
自然生成論（源質是世界自然產物）
↓ 被推翻
人工源質論（源質可能被人工製造）
```

## 核心問題鏈

```
林墨取得「失落源質殘頁」（承接卷一末）
↓
殘頁記載：第二環源質存在人工合成記錄
↓
進入深層礦區調查（Ch081）
↓
發現「不存在於資料庫」的礦石（Ch090）
↓
白塔研究院介入（Ch100）
↓
大型礦脈爭奪戰爆發（Ch120）
↓
林墨在爭奪戰中發現礦脈內部的人工結構（Ch140）
↓
地下都市入口開啟（Ch180～Ch200）
↓
發現殘缺週期表（卷末）
```

## 卷弧線

林墨：從「獨行研究者」→「被迫選邊站的資訊持有者」
