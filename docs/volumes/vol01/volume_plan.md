# 第一卷：月池之謎

## 卷主題

為什麼新手村的池水裡，會有不該存在的源質？

## 核心問題鏈

```
月靈源質異常（Ch001）
↓
古代符文浮現（Ch001）
↓
礦洞中發現夜光石含命運源質（Ch006）
↓
命運源質分布有規律（Ch010）
↓
老陳的暗示（Ch015）
↓
找到古代祭壇入口（Ch020）
↓
祭壇短暫啟動，第四環源質顯現（Ch080）
```

## 卷弧線

林墨：從「好奇的觀察者」→「意識到自己捲入更大事件的研究者」

- 月牙村（Ch001～Ch005）
- 月牙森林（Ch004～Ch012）
- 初級礦洞（Ch006～Ch025）
- 古代祭壇（Ch020～Ch080）
