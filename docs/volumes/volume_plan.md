# 《源質紀元》五卷總綱（研究生態系版本）

# 第一卷：月池之謎

章數：80 ~ 120章

核心主題：世界並非只有四元素。

世界認知

主流理論：

世界由土、水、火、風組成。

玩家認為：

源質只是材料系統。

主線事件

林墨發現月牙池存在未知源質。

透過超限解析能力找到月靈源質。

重要發現

第三環存在。
靈性源質存在。

勢力介入

青葉工坊
白塔研究院
黑石商會

卷末高潮

月池底部出現古代祭壇。

林墨獲得失落源質殘頁。

被推翻理論

四元素理論。

新理論

世界存在更高階源質。

# 第二卷：礦脈深層

章數：100 ~ 150章

核心主題：源質並非自然生成。

主線事件

主角進入深層礦區，發現不存在於資料庫的礦石。

重要發現

雷源質
冰源質
高純度源質結晶

勢力衝突

世界礦業聯盟
黑石商會
赤焰戰盟

卷末高潮

發現古代地下城市與殘缺週期表。

被推翻理論

源質來自自然。

新理論

源質可能被人工製造。

# 第三卷：失落文明

章數：120 ~ 180章

核心主題：世界曾存在超級文明。

主線事件

探索地下城市，破解古代文字。

重要發現

時間源質
空間源質
古代研究紀錄

勢力全面登場

命運教團
深淵煉成會
白塔研究院

卷末高潮

確認第一煉成者文明真實存在。

被推翻理論

源質週期表是自然形成。

新理論

週期表是人造知識。

# 第四卷：研究戰爭

章數：150 ~ 250章

古代資料庫崩解。

各大勢力爭奪研究成果。
命運源質正式曝光。
因果源質被驗證。
世界研究峰會召開。
研究成果遭大規模竊取。
源質共振事件爆發。
世界各地出現巨型符文節點。
林墨確認世界存在整體結構。

世界局勢

白塔研究院：研究成果應公開。
黑石商會：研究成果應資產化。
命運教團：第四環知識必須封印。
深淵煉成會：任何知識皆可使用。

重要發現

D-030 命運源質
D-031 因果源質
D-032 源質共振理論
D-033 世界級源質節點

卷末高潮

世界節點連接後形成巨大結構。

世界本身是一座超大型煉成陣。

林墨成長

源質學者 → 煉成大師。

# 第五卷：世界煉成陣

章數：200 ~ 300章

核心主題：誰在研究誰？

舊理論 → 最終真相

玩家在研究世界。

↓ 被推翻

世界也在研究玩家。

卷定位：世界級研究競賽終局

本卷不存在『主角單人破解世界』。

所有重大成果由不同勢力共同完成。

林墨的定位是：研究整合者（Integrator）。

世界研究聯盟成立

成員：

白塔研究院

部分玩家公會

非成員：

黑石商會
命運教團
深淵煉成會

世界研究排行榜

各勢力競爭：

第一個破解第五環。
第一個破解世界核心。
第一個完成終極公式。

重大研究成果

D-040 創造源質 —— 白塔研究院發現。
D-041 毀滅源質 —— 深淵煉成會發現。
D-042 秩序源質 —— 命運教團發現。

D-044 世界核心 —— 多方共同發現。

第七實驗室（主角團隊）

林墨建立跨勢力研究團隊：

林墨：解析者。
白塔叛逃學者：理論建構。
礦業聯盟勘探師：材料研究。
前黑石配方師：公式優化。
命運教團流亡祭司：第四環研究。

真正的關鍵發現

所有勢力各自掌握一塊拼圖：

深淵：毀滅。
命運：秩序。
黑石：混沌。
